Introduction to NumPy (Python) 
NumPy (Numerical Python) is a core Python library used for numerical computing, 
especially in Data Science, Machine Learning, AI, and scientific computing.

What is NumPy?

NumPy provides:
1.A powerful N-dimensional array object (ndarray)
2.Fast mathematical and statistical operations
3.Tools for linear algebra, random numbers, and Fourier transforms
